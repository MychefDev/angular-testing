Welcome to the MultiTool App!

Select an option from the menu above to get started.

This project is designed to facilitate unit testing, integration testing, and end-to-end (E2E) testing for various utilities. Each tool included in this application serves as a functional example to demonstrate the importance of testing in software development.

You can explore features like the To-Do List, Calculator, and Chronometer, all of which are structured to be tested individually and as part of the entire application.

We encourage you to try out all types of tests where you think it is necessary to ensure the functionality and reliability of the application.
